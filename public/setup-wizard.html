KONTO & STRUKTUR
Meta Business Manager
Företagskonto som samlar alla Meta-tillgångar.
KRÄVS ▼
— SKAPA BUSINESS MANAGER
  • 1Gå till business.facebook.com → klicka "Skapa konto".
  • 2Logga in med privat Facebook-konto (syns inte för kunder).
  • 3Fyll i företagsnamn och e-post → verifiera via mail.
→ Meta Business Manager
Annonskonto (Ad Account)
Aktivt annonskonto kopplat till ditt BM.
KRÄVS ▼
— LÄGG TILL ANNONSKONTO
  • 1BM → Inställningar → Annonskonton → Lägg till.
  • 2Välj "Skapa nytt" eller koppla befintligt.
  • 3Sätt valuta SEK och tidszon Stockholm.
  • 4Lägg till betalningsmetod under Fakturering.
→ Annonskonton i BM
Facebook-sida kopplad
Behövs för annonser och page insights.
REKOMMENDERAS ▼
— KOPPLA FACEBOOK-SIDA
  • 1BM → Inställningar → Facebook-sidor → Lägg till.
  • 2Välj "Lägg till en sida" och sök efter ditt företag.
  • 3Bekräfta äganderätt via e-post eller sidadmin.
→ Facebook-sidor i BM
API & ÅTKOMST
Meta App skapad
Developer Portal — Marketing API aktiverat.
KRÄVS ▼
— SKAPA META APP
  • 1Gå till developers.facebook.com → My Apps → Create App.
  • 2Välj typ "Business", namnge appen.
  • 3Vänstermeny → Add Products → Marketing API → Set Up.
  • 4Settings → Basic → lägg till fabricken.se i App Domains → Spara.
→ Meta Developer Portal
Access Token genererad
Graph API Explorer — expirerar 1–2h, skicka direkt!
KRÄVS ▼
— GENERERA ACCESS TOKEN
  • 1Öppna Graph API Explorer → välj din app.
  • 2Välj User Token → klicka "Generate Access Token".
  • 3Kryssa i: ads_management, ads_read, business_management, pages_read_engagement.
  • 4Kopiera token och klistra in i Steg 2 nedan — sedan skickar du med ett klick.
→ Graph API Explorer
SPÅRNING & PIXEL
Meta Pixel installerad
Spårar konverteringar — ROAS och retargeting.
REKOMMENDERAS ▼
— INSTALLERA META PIXEL
  • 1BM → Events Manager → Datakällor → Lägg till.
  • 2Välj "Meta Pixel" → ge det ett namn.
  • 3Installera via Google Tag Manager eller manuell kod på din webbplats.
  • 4Verifiera med Meta Pixel Helper (Chrome-tillägg).
→ Events Manager
Conversions API (CAPI)
Server-side spårning — täcker det ad-blockers döljer.
VALFRITT ▼
— AKTIVERA CAPI
  • 1Events Manager → välj din Pixel → Inställningar.
  • 2Under "Conversions API" → klicka Kom igång.
  • 3Välj partner-integration (Shopify, GTM etc.) eller manuell API.
→ Events Manager
